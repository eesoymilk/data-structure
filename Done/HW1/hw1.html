  1. (10%) Using the ADT1.1 NaturalNumber in the textbook pp.10, add the following operations to the NaturalNumber ADT: Predecessor, IsGreater, Multiply, Divide.
    Sol:
    Predecessor(x) ::= if (x == 0) Predecessor = 0
    else Predecessor = x - 1
    IsGreater(x, y) ::= if ((x - y) <= 0) IsGreater = false
    else IsGreater = true
    Multiply(x, y) ::= if (y == 0) Multiply = 0
    else if (y == 1) Multiply = x
    else Multiply = x + Multiply(x, y - 1)
    Divide(x, y) ::= if (y == 0) Divide = ERROR
    else if (x == 0) Divide = 0
    else if (x < y) Divide = 0
    else Divide = 1 + Divide(x – y, y)
